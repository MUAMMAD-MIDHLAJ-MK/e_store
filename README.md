# 🛍️ Luxora - Premium Django E-Commerce Website

A modern and responsive e-commerce web application built with **Django**, **Python**, **HTML5**, **CSS3**, and **JavaScript**. This project provides a premium shopping experience with a clean user interface and essential e-commerce features.

---

- 🏠 Premium Home Page
- 🛍️ Product Listing
- 📂 Category Management
- 🔍 Product Search
- ❤️ Wishlist UI
- 🛒 Shopping Cart UI
- 👤 User Authentication
- 📱 Fully Responsive Design

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/e_store.git
```

### Go to Project Folder

```bash
cd e_store
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
